# CDM / Classroom Dark Mode
A stylish, minimalist dark theme for **Google Classroom**.

## Installation:

1. Download the **Stylus** browser extension.

2. Go to this link [CMD Style](https://userstyles.world/api/style/25373.user.css) and click **Install style**.

3. Done! You can now use it.
